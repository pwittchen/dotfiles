dotfiles
========
my dotfiles used on Linux and macOS

installation
------------

```shell
git clone git@github.com:pwittchen/dotfiles.git
cd dotfiles
./install.sh
```

update
------

```shell
cd dotfiles
git pull
./install.sh
```

git setup
---------
- git for work: `setupGitForWork`
- git for personal projects: `setupGitPersonal`

Default config is personal.
